让孩子看见自己的价值

不是夸他，是帮他看见。

我是在评价孩子，还是在帮孩子看见自己的价值？

这是价值四圈的核心问题。每次回应孩子之前，先问自己这一句。

点击每一圈，看看孩子已经做到了什么

常见错误动作

点击卡片翻转查看解释和替代方案

空洞夸奖
点击查看 →
"你真棒！""你真聪明！""你好厉害！"
没有事实支撑的夸奖听起来假，孩子会免疫。改为：描述具体行为 + 价值命名。
条件式价值
点击查看 →
"你考好了，真棒！""你听话，才是个好孩子。"
价值感和表现绑定，一旦表现不好，价值感就崩塌。改为：就事论事，不贴标签。
跳过事实评价
点击查看 →
"你是个好孩子。"（没有事实支撑）
价值判断必须从具体事实出发，否则是空中楼阁。改为：先描述事实，再命名价值。
价值加压
点击查看 →
"以后都要这样！""你得保持住啊！"
夸奖变成压力，孩子下次反而不敢做了。改为：描述当下，然后留空间。
忽视微小变化
点击查看 →
"才做这么一点。""这有什么好说的。"
微小的变化才是真正的进步。改为：看见每一点，然后说出来。
比较抹杀
点击查看 →
"你看人家XXX。""你怎么不如人家。"
比较是价值感的天敌。改为：只和昨天的自己比，每一次都是赢。
价值反转
点击查看 →
"做得不错，但是……""好是好，不过……"
一句"但是"毁掉所有努力。改为：去掉"但是"，或换成"同时"。"不错，同时……"

话术工具箱

事实描述 + 价值命名，❌→✅ 对比

孩子自己收拾了房间，你脱口而出："真棒！你真是个好孩子！"
"你把书放回了书架，笔收到了笔筒里——这叫自律，是你自己管住了自己。"
"你帮妈妈拿东西了，真乖！"（条件式 + 空洞）
"你帮妈妈把碗端到了桌上——你做这件事让家里好走了一步，这是家庭圈的贡献。"
"这次考得好，妈妈太高兴了！"（把孩子的价值绑在父母的情绪上）
"你这道题之前不会，现在做对了——你自己搞定了一道难题，这是你自己圈的能力。"
"以后每天都要自己收拾！"（价值加压）
"你今天自己收了书桌——这说明你可以管好自己的地方。你觉得明天会怎样？"
"你看你，又忘了。" "才放好又乱了。"（否定微小变化）
"今天你放好了5分钟——比昨天多放了3分钟。你自己发现了吗？"
"你看人家小明每天都自己整理。"（比较抹杀）
"你自己把衣服叠好了——这跟别人没关系，这是你自己做到的。"

提问引导话术

用1-2个问题，引发孩子自己思考，不要直接给答案

"你自己把这件事做完了，你觉得这是你身上的什么能力？"
引导孩子自己命名，而不是你告诉他
"你今天帮了妈妈，你觉得家里因为你好在哪里？"
让孩子连接到家庭圈的价值
"你自己管住了玩手机——你觉得这叫什么？"
让孩子自己说出"自律"这个词
"你帮同学讲了一道题——你觉得这对别人来说意味着什么？"
连接到社会圈，让孩子感受自己的影响力
